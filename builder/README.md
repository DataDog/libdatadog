# builder

Build tool for libdatadog project.

## Overview

The `builder` crate provides build automation and tooling for the libdatadog project, including cross-compilation, packaging, and release management.

## Features

- **Cross-compilation**: Build for multiple platforms
- **Package Generation**: Create distribution packages
- **Release Automation**: Automate release processes
- **Build Configuration**: Manage build configurations
- **Artifact Management**: Handle build artifacts

## Usage

```bash
cargo run --package builder -- <command>
```

# libdd-profiling-replayer

Profile replay tool for testing and development.

## Overview

`libdd-profiling-replayer` provides utilities for replaying captured profiling data, useful for testing, debugging, and performance analysis of profiling pipelines.

## Features

- **Profile Replay**: Replay previously captured profiles
- **Testing**: Test profiling pipelines with realistic data
- **Performance Analysis**: Analyze profiling overhead and performance
- **Format Conversion**: Convert between profiling formats
- **Validation**: Validate profile data integrity

## Use Cases

- Testing profile exporters
- Debugging profile processing
- Performance benchmarking
- Format validation
- Load testing profiling backends

# libdd-sidecar-macros

Procedural macros for libdd-sidecar.

## Overview

`libdd-sidecar-macros` provides procedural macros used by `libdd-sidecar` for code generation and compile-time utilities.

## Macros

The crate provides procedural macros for:
- IPC interface generation
- Serialization/deserialization helpers
- Code generation for sidecar services

## Usage

This crate is typically used as an implementation detail of `libdd-sidecar` and not used directly.
